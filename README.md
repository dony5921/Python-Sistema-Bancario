# 💰 Sistema Bancário em Python

Sistema bancário simples desenvolvido em Python, executado via terminal, com funcionalidades de gerenciamento de usuários, contas e operações financeiras.

##  Funcionalidades

* 💵 Depósito de valores
* 💸 Saque com limite diário
* 📄 Exibição de extrato
* 👤 Criação de usuários
* 🏦 Criação de contas bancárias
* 📋 Listagem de contas

##  Regras de negócio

* Limite de saque por operação
* Limite diário de saques
* Não permite saque sem saldo
* Não permite valores inválidos

##  Tecnologias

* Python
* Execução via terminal (CLI)

## ▶️ Como executar

```bash
git clone https://github.com/dony5921/Python-Sistema-Bancario.git
cd Python-Sistema-Bancario
python sbancario.py
```

##  Menu do sistema

```text
[d] Depositar
[s] Sacar
[e] Extrato
[nc] Nova conta
[lc] Listar contas
[nu] Novo usuário
[q] Sair
```

## Estrutura do código

* Funções separadas para cada operação (depositar, sacar, extrato)
* Controle de usuários e contas via listas
* Loop principal controlando o fluxo do sistema

##  Objetivo

Projeto desenvolvido para prática de lógica de programação, manipulação de dados e simulação de regras de negócio de um sistema bancário.
